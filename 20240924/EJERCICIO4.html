HTML

Lenguaje de Marcas de Hipertexto, del inglés HyperText Markup Language

Objetivo HTML

El HTML es un lenguaje de programación que permite mostrar archivos de forma estructurada. De igual forma permite incluir imágenes, gráficos, sonidos, vídeos y otro tipo de archivos multimedia

Características HTML

HTML es un lenguaje de marcado que nos permite indicar la estructura de nuestro documento mediante etiquetas

Etiquetas HTML

Las etiquetas HTML contienen instrucciones para definir un elemento HTML específico e indican cómo mostrarlo en la página
