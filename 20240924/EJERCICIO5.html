Etiquetas heade en HTML

<h1>,<h2>,<h3>,<h4>,<h5>,<h6> establecen encabezados: marcas que identifican la importancia del siguiente trozo de texto.

Primer nivel

Se utiliza la etiqueta <h1> para el primer nivel de encabezado

Segundo nivel

Se utiliza la etiqueta <h2> para el segundo nivel de encabezado

Tercer nivel

Se utiliza la etiqueta <h3> para el tercer nivel de encabezado
